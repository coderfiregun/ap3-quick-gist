# Hands-on High Level System Design (HHLD) Course

## Course Snapshot
- **Format:** 8 weeks, 3 live classes per week, 2 hours per class
- **Access:** Lifetime recordings, code repositories, and all future live batches
- **Support:** Discord community and dedicated doubt resolution after every session
- **Focus:** Bridging HLD concepts with full-stack implementation across MERN and DevOps stacks

## Week-by-Week Breakdown

### Week 1 – AWS & Platform Foundations
- Provisioning compute with EC2 and securing access with IAM and security groups
- Networking setup via VPC, subnets, and routing
- Managed data stores: DynamoDB, RDS, and S3
- Global delivery with CloudFront and DNS management using Route 53
- Event streaming through MSK (Managed Kafka)
- Containerization basics with Docker and orchestration via Kubernetes
- Connecting cloud components through Node.js services

### Weeks 2–3 – Messaging Platform (WhatsApp-Style)
- HLD deep dive for large-scale chat systems
- Designing microservices for messaging, presence, media, and notifications
- Integrating persistence, caching, and real-time transport
- Deploying services with CI/CD pipelines and observability hooks

### Weeks 4–5 – Video Streaming Platform (YouTube-Style)
- End-to-end architecture for video ingest, encoding, storage, and delivery
- Implementing metadata services, recommendation flows, and content delivery
- Handling large file uploads, CDN distribution, and playback analytics
- Deploying the ecosystem with scalability and resilience guardrails

### Weeks 6–7 – Stock Brokerage Platform (Zerodha-Style)
- Requirements gathering and domain modeling for trading workflows
- Building order management, quotes, risk, and portfolio microservices
- Ensuring low-latency execution, consistency, and regulatory compliance
- Integrating third-party market data feeds and notification pipelines

### Week 8 – Common System Components
- Centralized logging, metrics, and distributed tracing
- Alerting strategies and incident response runbooks
- Infrastructure cost tracking and optimization playbook
- Final Q&A and capstone review

## Deliverables & Resources
- Production-grade reference implementations for each project
- Infrastructure-as-code snippets and deployment templates
- Reusable microservice scaffolds and Node.js integration patterns
- Lifetime access to updated content, mentor sessions, and community discussions
